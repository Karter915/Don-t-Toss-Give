# Don-t-Toss-Give
Experimental project working with a javascript backend server and a HTML frontend. Proof of concept

Initial Description:
The overall goal of this project is to reduce food waste on the Penn state Campus. This is to be achieved by a web application that will allow event holders to advertise left over food for students to scheduler a pick up for. 

User will need to install node.js from https://nodejs.org/en
This will give access to a node module which is used to run the program
